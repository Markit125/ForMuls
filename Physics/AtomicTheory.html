Параметры элемента

A — Атомная масса/Число нуклонов
Z — Заряд ядра/Число протонов/электронов
A-Z — Число нейтронов
X — Название элемента

Массы составных частей атома

mпротона = mp = 1,00728 а.е.м.
mнейтрона = mn = 1,00867 а.е.м.
mэлектрона = me = 0,0005486 а.е.м.

Масса атома

mядра = mатома - Z·me

Дефект масс

Чтобы найти дефект масс:

  1. Суммируем массы протонов и нейтронов ядра (m1)
  2. Вычитаем из массы атома, взятой из табличных данных, массу всех электронов атома (m2)
  3. Находим разность этих масс (m1 - m2)

m1 = Z·mp + (A-Z)·mn

m2 = mатома - Z·me

Δm = m1 - m2 = Z·mp + (A-Z)·mn - mатома + Z·me
