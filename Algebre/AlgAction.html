Действия над алгебраическими выражениями

Абсолютная величина числа

Если a ≥ 0, то |a| = a; если a < 0, то |a| = -a

Действия над многочленами

(a + b + c) = ax + bx + cx

(a + b + c)(m + m) = a(m + n) + b(m + n) + c(m + n) = am + an + bm + bn + cm + cn

Действия над дробями

Формулы сокращённого умножения

(a ± b)2 = a2 ± 2ab+b2

(a ± b)3 = a3 ± 3a2b + 3ab2 ± b3

a2 - b2 = (a + b)(a - b)

a3 + b3 = (a + b)(a2 - ab + b2)

a3 - b3 = (a - b)(a2 + ab + b2)

am- bm = (a - b)(am - 1 + am - 2b +...+ abm - 2 + bm - 1)

(a + b + c)2 = a2 + b2 + c2 + 2ab + 2ac +2bc

Действия со степенями

aman = am + n

am:an = am - n

(ab)m = aman

(am)n = amn

a0 = 1, a ≠ 0
